• 注意：a.所有页面打乱坐标均为还原坐标；b.编码为彳亍编码（后期会支持自定义编码）；c.暂不设置cookie，所有输入内容请自行保存。
• 网站开发的主要目的是生成贴近实战的打乱公式，从而更高效地进行特定内容的练习。目前主要功能是棱块公式训练、角块公式训练、读码还原助手，其他页面陆续优化新增中。

1.读码还原助手

• 主要功能为两项：a.输入打乱，输出盲拧编码；b.输入盲拧编码，输出魔方状态（并有3D魔方显示）。
• 此页面主要为了解决初学者编码容易编错的问题，最初是为了提供一个比APP通用性更好的读码工具。 后来又加入了还原功能，可以尝试不同的编码方法，如果3D魔方最后被还原了则说明编码正确，是一个入门时能确定编码是否有问题的工具。 初学者复盘或者遇到问题时，还可以每做一条公式就同步输入还原编码，核对3D魔方的状态是否一致，从而快速定位问题。
• 默认缓冲为棱A(UF)角J(UFR)，修改缓冲块时，需要注意调整借位位置，借位位置与缓冲位置不能重复。
• 保持色相借位根据教程选择即可，跳编法（即固定借位法），初学者不需要选。

2.棱块公式训练：

• 主要功能为输入需要练习的公式、输出比较贴近实战的打乱。
a. 精准生成模式：从输入训练公式中优先选取出现次数较少的公式，构造状态并求解打乱公式（生成的打乱没有小循环），当输入训练公式全部出现至少一次则停止生成并输出打乱及信息。
b. 随机生成模式：随机生成打乱并读码，从中挑选出包含较多输入训练公式的打乱。每次点击生成随机生成1万条打乱，最终输出训练公式数量排名较前的100条打乱。

3.角块公式训练：

• 功能同棱块训练。

4.翻角训练：

• 无奇偶，角块会出现除缓冲块之外的两个翻角，方向任意，棱块为还原状态。

5.棱块浮动训练：

• 【浮动顺序】目前仅支持输入两个字母，第一位为主缓冲编码，第二位为副缓冲编码。 脚本会从输入副缓冲公式中优先选取出现次数较少的公式，直到所有副缓冲公式全部出现至少一次则停止生成并输出打乱及信息。
• 【排除位置】为不会被打乱的棱块位置。 比如用户已经会了4个缓冲并且顺序是AEGC，现在想练习第三第四缓冲，则在【排除位置】中输入AE，在【浮动顺序】中输入GC即可。

6.奇偶带翻训练：

• 输入奇偶带翻编码（第一位为奇偶码，第二位为翻色码），生成相同数量的随机打乱，所有输入的奇偶带翻情况均会出现1次，棱块随机打乱。
