三盲训练器的公测版本(V-0.2)

棱块训练：随机生成模式为随机生成打乱并从中挑选出尽量多符合训练集公式的打乱，每次点击生成随机生成1万条打乱，最终输出符合公式数量排名较前的100条打乱。

奇偶带翻：输入n组LTCT编码，生成随机的n条公式，角块一定会遍历所有输入LTCT的情况，棱块完全随机。

浮动缓冲：目前仅支持棱块。输入主缓冲和副缓冲，生成的所有打乱状态会将副缓冲公式遍历一遍。如果对奇偶状态下浮动原则不确定，建议选择无奇偶状态。

翻角训练：无奇偶，棱块还原状态，角块会出现除缓冲块之外的两个翻角，方向任意。
